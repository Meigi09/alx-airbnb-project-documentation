---

## ðŸ§© Key Features

### 1. User Authentication & Management
- Signup, login, and logout
- Password hashing and secure storage
- Role-based access control (guest, host, admin)
- User profile management

### 2. Property Management
- CRUD operations for properties
- Property details: name, description, location, price, images
- Host-specific property listings

### 3. Booking System
- Create, update, cancel bookings
- Check property availability
- Booking status: pending, confirmed, canceled

### 4. Payments
- Record payments linked to bookings
- Multiple payment methods: credit card, PayPal, Stripe
- Track payment dates and amounts

### 5. Reviews & Ratings
- Guests can leave reviews and ratings for properties
- Aggregate property ratings for display

### 6. Messaging System
- Guest-host communication
- Messages stored with sender, recipient, and timestamp

### 7. Admin Controls
- Manage users, properties, bookings, and reviews
- Generate reports and monitor system activity
